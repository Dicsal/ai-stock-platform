# 升級版 AI 選股平台

新增功能：
- 📈 K線圖 + SMA 技術線
- 🧠 AI 選股打分排序系統

## 本地執行
```bash
pip install -r requirements.txt
streamlit run main.py
```

## Streamlit Cloud 部署
1. 上傳此專案到 GitHub
2. 在 https://streamlit.io/cloud 設定：
   - repo: yourusername/ai-stock-platform
   - branch: main
   - main file: main.py
3. 點 Deploy
